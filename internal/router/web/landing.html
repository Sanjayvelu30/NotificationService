Notification Engine Monolith

An enterprise-grade Go notification scheduler demonstrating advanced transactional database locking, fault-tolerant worker pools, and secure multi-tenant isolation.

🚀 SKIP LOCKED Concurrency
Utilizes PostgreSQL `SKIP LOCKED` transactional locking queries to safely scale poller worker threads without double-delivery risks.
🔄 Exponential Backoff
Scales delivery retries automatically with configurable wait timings ($2^{\text{retry\_count}}$ sec) to protect API resources from flood traffic.
🔒 Auth0 Identity Security
Protects downstream API endpoints using secure Auth0 user credentials and token validation caches built in-memory.
⚡ User-Scoped Rate Limiter
Shields resources from account abuse using custom fixed-window rate limiting scoped directly to your validated user ID.
📁 Customizable Templates
Saves template modifications directly to PostgreSQL, isolated securely per tenant using composite primary key indexing constraints.
📩 Resend Email Integration
Wired directly to the Resend SMTP API to dispatch production-grade emails with template variable parsing.
🛠 Dead Letter Queue (DLQ)
Quarantines failed deliveries after max retry exhaustion, recording execution diagnostics for developer inspection.
📊 Live Dashboard Console
Streams live background engine logs and dynamic API quota gauges straight to your browser developer console dashboard.
